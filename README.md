# IT管理部智能助手

钉钉FAQ机器人，一键启动！

## 快速启动

### Windows用户
双击运行 `启动机器人.bat`

### Mac/Linux用户
```bash
bash 启动机器人.sh
```

## 启动后

1. 服务自动运行在 http://localhost
2. 确保 natapp 隧道正常运行（如果有）
3. 在钉钉中 @IT管理部智能助手 提问

## 注意事项

- 首次运行会自动安装依赖（约1-2分钟）
- 保持电脑开机，机器人才能正常运行
- 关闭命令行窗口可停止服务

## 常见问题

**Q: 提示"Python未安装"**
A: 下载安装Python：https://www.python.org/downloads/

**Q: 机器人没有响应**
A: 检查natapp是否正常运行
