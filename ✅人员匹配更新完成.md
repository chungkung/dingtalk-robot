## 📝 技术实现

### 匹配算法

```python
def _calculate_score_v2(self, question: str, person: Dict) -> float:
    """快速计算匹配得分"""
    person_keywords = person.get('keywords', '').split(',')
    person_keywords = [kw.strip().lower() for kw in person_keywords]
    
    # 精确匹配得分（关键词直接出现在问题中）
    exact_matches = sum(1 for kw in person_keywords if kw in question)
    
    # 模糊匹配得分（部分匹配）
    fuzzy_matches = sum(0.5 for kw in person_keywords 
                       if any(k in question for k in kw.split()))
    
    # 综合得分
    total_score = (exact_matches * 1.0 + fuzzy_matches * 0.5) / max(len(person_keywords), 1)
    return min(total_score, 1.0)
```
